# Proyecto Django prueba (con diseño integrado)

Proyecto para la materia de Programación Web con Django.
Incluye el diseño base proporcionado (CSS/JS) y las funcionalidades:

- App `inicio` con página de registro usando plantilla de diseño
- App `registros` con alumnos, comentarios, archivos
- CRUD de comentarios de contacto
- Carga de archivos
- Consultas ORM y SQL
- Admin personalizado básico
